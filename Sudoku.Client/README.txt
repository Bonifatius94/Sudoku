
=========================================================
                  SUDOKU SOLVER v1
=========================================================

Features:
========

Feature 1: Generate sudokus (automatic mode), try to solve them, see the solution by making the program solve it

Feature 2: Type in a sudoku from other source (manual mode) and make the program solve it / try to solve it yourself and see the solution
